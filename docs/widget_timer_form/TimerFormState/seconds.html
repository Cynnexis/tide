void seconds=(int s)

The second field of the timer.

The given value must be in the range 0 ; 59. Setting a new value will trigger the widget.onChanged callback.

Implementation

set seconds(int s) {
  if (s != _seconds) {
    if (s < 0 || 59 < s) {
      throw ArgumentError("Expected second to be in [0 ; 59], got $s.");
    }

    _seconds = s;
    onChanged();
  }
}
