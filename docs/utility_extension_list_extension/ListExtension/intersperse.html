Add the element item between each element of this list.

This function returns nothing, the actual list will be modified. If you want to construct a new list without affecting this instance, please use intersperseCopy instead.

Note that the current list needs to be modifiable and growable, or an UnsupportedError will be thrown.

Source code inspired from https://stackoverflow.com/a/5921708 by senderle (consulted on April 9th, 2022).

void intersperse(final T item) {
  if (length <= 1) {
    return;
  }

  for (int i = 1; i < length; i += 2) {
    insert(i, item);
  }
}
